# apps/filebrowser

Modul-Skeleton (Schritt 1 der Modularisierung).

Dieses Verzeichnis wird in den folgenden Commits befuellt:
- compose.yml (Spiegel der bestehenden compose.*.yml-Definitionen)
- .env.example (App-spezifische Variablen, nur Platzhalter)
- backup.include (Pfade fuer scripts/backup.sh)
- healthcheck.sh
- caddy.disabled / caddy.authentik.disabled (Snippets)

Bestehender Betrieb laeuft unveraendert ueber die compose.*.yml im Repo-Root.
